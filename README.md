# Breezie - 情绪管理助手

一个帮助用户管理情绪、进行情绪疏导对话的应用。

## 功能特点

### ☁️ Breezie特色
- 云朵样子的毛茸茸标志，温暖友好的视觉设计
- 智能情绪陪伴，提供贴心的情绪管理支持
- 简洁直观的界面，让情绪管理变得轻松

### 🎯 情绪选择
- 提供7种情绪状态：愤怒、厌恶、恐惧、快乐、悲伤、惊讶、复杂
- 复杂情绪包含焦虑、嫉妒、尴尬等复合情绪
- 直观的卡片式选择界面
- 每种情绪都有对应的图标和描述

### 💬 智能对话
- 基于选择情绪进行个性化对话
- Breezie智能助手进行情绪疏导
- 实时聊天界面，支持消息历史记录
- 自动保存对话会话

### 📊 情绪轨迹
- 记录和追踪情绪变化
- 提供情绪统计和分析
- 情绪轨迹曲线图和线性回归趋势线
- 可视化情绪分布图表
- 查看历史情绪记录
- 概览页面集成图表展示

### 💾 数据持久化
- 使用Zustand进行状态管理
- 自动保存到浏览器本地存储
- 无需注册账号，数据本地保存
- 简洁的本地存储架构，专注于核心功能

## 技术栈

- **前端框架**: Next.js 15 (App Router)
- **UI组件**: shadcn/ui + Tailwind CSS
- **状态管理**: Zustand
- **图标**: Lucide React
- **通知**: Sonner
- **日期处理**: date-fns
- **图表**: Recharts

## 快速开始

1. 安装依赖：
```bash
npm install
```

2. 启动开发服务器：
```bash
npm run dev
```

3. 打开浏览器访问 [http://localhost:3000](http://localhost:3000)

## 使用说明

### 开始对话
1. 在首页选择当前的情绪状态
2. 点击对应的情绪卡片进入对话界面
3. 与Breezie进行对话，获得情绪疏导

### 查看情绪轨迹
1. 点击右上角的"轨迹"按钮
2. 查看情绪统计和分布
3. 浏览历史情绪记录

### 数据管理
- 所有数据自动保存在浏览器本地存储中
- 清除浏览器数据会删除所有记录
- 无需担心隐私问题，数据不会上传到服务器
- 简化的架构，专注于核心的情绪管理功能

## 开发说明

这是一个纯前端演示项目，主要用于展示情绪管理应用的基本功能。Breezie对话功能使用预设的回复模板，在实际应用中可以集成真实的AI服务。
