# 🚀 Breezie Gemini 2.5 升级指南

## 📋 升级概述

Breezie已成功升级到Google Gemini 2.5模型，这是目前最先进的AI模型之一，提供更智能、更自然的对话体验。

## 🔄 主要变更

### 1. **模型升级**
- **之前**: `gemini-1.5-flash`
- **现在**: `gemini-2.0-flash-exp` (Gemini 2.5 实验版)

### 2. **性能提升**
- **最大Token数**: 从1200提升到1500
- **温度设置**: 从0.8优化到0.85
- **Top-P**: 从0.9提升到0.95
- **Top-K**: 从40提升到50

### 3. **动态Token分配优化**
- **高参与度**: 最高1800 tokens (之前1200)
- **中等参与度**: 最高1200 tokens (之前900)
- **标准参与度**: 800 tokens (之前600)

### 4. **安全设置增强**
- 添加了完整的安全过滤设置
- 支持内容安全策略
- 增强的滥用防护

## 🎯 Gemini 2.5 新特性

### **语言理解能力**
- 更自然的对话流程
- 更好的上下文理解
- 更准确的意图识别

### **情感支持增强**
- 更温暖、更人性化的回应
- 更好的情感共鸣
- 更个性化的支持

### **响应质量**
- 更长的、更有意义的回应
- 更好的逻辑连贯性
- 更丰富的表达方式

## ⚙️ 技术配置

### **API端点**
```
https://generativelanguage.googleapis.com/v1beta/models/gemini-2.0-flash-exp:generateContent
```

### **请求配置**
```typescript
{
  model: 'gemini-2.0-flash-exp',
  maxTokens: 1500,
  temperature: 0.85,
  topP: 0.95,
  topK: 50,
  safetySettings: [...],
  candidateCount: 1
}
```

### **超时设置**
- **请求超时**: 20秒 (之前15秒)
- **重试机制**: 自动错误处理
